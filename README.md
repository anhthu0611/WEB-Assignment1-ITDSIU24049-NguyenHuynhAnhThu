# Project Description

This project is a personal website about a coffee shop, used for practicing Web Application Development.

# Web Structure

Web Pages/ 
- HTML and JSP pages (home.html, about.html, contact.html, menu.html, post-1.html, post-2.html, order.jsp)

- assets/ (images, fonts, and style.css)

Source Packages/ (Ordering.java)

WEB-INF/ Configuration files for the web application

# How to Run the Project

Requirements:

- Java JDK

- Apache NetBeans

- Apache Tomcat

1. Open the project in NetBeans.

2. Click Run Project.

3. NetBeans will start the Tomcat server automatically.

4. Open the browser and go to:

http://localhost:8080/Assignment1_Netbeans/
